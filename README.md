# Cinema-Garden

## 1. 메인 페이지

    - top 5 영화로 header 캐러셀 적용
    - 상단으로 이동 버튼
    - 검색 기능
        - submit으로 enter / click 적용
    - 영화 카드
        - grid
    - 카드 필터링
        - 인기(popularity) / 평점(vote_average) / 이름(title)
        - 개봉일
        - 평점은 내림차순 정렬
    - 상영 예봉작 티저/예고편 iframe으로 추가
    - footer
        - 팀원 블로그/깃 아이콘
    - 상단으로 이동 버튼
